# Finance-Dashboard-Power-BI
Interactive Power BI dashboard analyzing sales, profit, and financial trends with dynamic filters and KPI insights.

## 📌 Overview

An interactive Power BI dashboard designed to track and analyze key financial metrics, enabling quick insights into sales performance, profitability, and business trends.

## 🛠 Tools Used

* Power BI (DAX, Power Query)

## 📊 Key Metrics

* Total Sales
* Total Profit
* COGS (Cost of Goods Sold)
* Units Sold
* Discounts

## 🔍 Key Features

* Interactive slicers for year, quarter, and country filtering
* KPI cards for quick performance overview
* Monthly sales trend analysis
* Product and segment-level breakdown
* Clean and user-friendly dashboard design

## 📈 Insights

* Identifies peak sales periods and seasonal trends
* Highlights top-performing products and segments
* Enables comparison of profitability across regions
